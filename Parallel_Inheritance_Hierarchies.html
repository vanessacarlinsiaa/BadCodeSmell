Parallel Inheritance Hierarchies

Parallel inheritance hierarchy adalah salah satu jenis "code smell" yang terjadi ketika terdapat dua atau lebih hierarki class yang sejajar atau sejajar dalam sebuah sistem. Ini terjadi ketika class-class dalam hierarki memiliki struktur yang serupa tetapi tidak berbagi hubungan warisan. Misalnya, ada hierarki class untuk tipe hewan dan hierarki class terpisah untuk jenis makanan, di mana class-class dalam hierarki makanan berhubungan dengan class-class dalam hierarki hewan, tetapi keduanya memiliki struktur terpisah.

Alasan utama terjadinya
"Parallel Inheritance Hierarchies":

  • Ketergantungan antar hirarki
  • Tidak menggunakan pola desain yang tepat
  • Pemecahan function yang berlebihan
  • Hubungan antar class yang tidak diatur dengan benar

Cara Mengatasi

  • Menggunakan pola desain, seperti pola Composite atau pola Strategy, untuk mengelompokkan dan memisahkan hierarki class.

  • Refactoring untuk menggabungkan hierarki-hierarki tersebut menjadi satu jika memungkinkan.
Sebelum Refactor
                            
// Hierarki class hewan
class Animal {
    void eat() {
        System.out.println("Animal is eating");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Dog is barking");
    }
}

// Hierarki class makanan
class Food {
    void cook() {
        System.out.println("Food is being cooked");
    }
}

class Meat extends Food {
    void grill() {
        System.out.println("Meat is being grilled");
    }   
}
Setelah Refactor
                            
// Menggabungkan hierarki class hewan dan makanan
class Animal {
    void eat() {
        System.out.println("Animal is eating");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Dog is barking");
    }

    // Dog dapat makan makanan apa saja
    void eatFood(Food food) {
        System.out.println("Dog is eating " + food.getClass().getSimpleName());
    }
}

// Hierarki class makanan
class Food {
    void cook() {
        System.out.println("Food is being cooked");
    }
}

class Meat extends Food {
    void grill() {
        System.out.println("Meat is being grilled");
    }
}

// Menggabungkan hierarki class makanan
class Meat extends Food {
    void grill() {
        System.out.println("Meat is being grilled");
    }
}

class Veggie extends Food {
    void chop() {
        System.out.println("Vegetable is being chopped");
    }
}

Dalam penyelesaian ini, masalah "Parallel Inheritance Hierarchy" diatasi dengan menggabungkan class makanan ke dalam hierarki class hewan. Selanjutnya, setiap jenis hewan dapat mengonsumsi jenis makanan apa pun sesuai kebutuhan. Hierarki class hewan dan makanan tidak lagi berjalan secara paralel, yang mengurangi redundansi dan meningkatkan struktur keseluruhan code.
