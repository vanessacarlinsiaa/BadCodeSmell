Change Preventers

Change preventers adalah smell yang mengartikan bahwa jika Anda perlu mengubah sesuatu dalam code anda, maka Anda harus membuat banyak perubahan di tempat lain juga. Pengembangan program jauh lebih rumit, sulit serta biayanya jauh lebih mahal sebagai hasilnya.

Divergent Change

Divergent Change adalah salah satu dari "Bad Code Smells" yang menandakan bahwa suatu bagian code harus diubah di banyak tempat yang berbeda setiap kali ada perubahan pada fitur atau function tertentu. Ini menunjukkan kurangnya pemisahan tugas dan ketergantungan yang tinggi di antara komponen-komponen code.

Parallel Inheritance Hierarchies

Parallel inheritance hierarchy adalah salah satu jenis "code smell" yang terjadi ketika terdapat dua atau lebih hierarki kelas yang sejajar atau sejajar dalam sebuah sistem. Ini terjadi ketika kelas-kelas dalam hierarki memiliki struktur yang serupa tetapi tidak berbagi hubungan warisan. Misalnya, ada hierarki kelas untuk tipe hewan dan hierarki kelas terpisah untuk jenis makanan, di mana kelas-kelas dalam hierarki makanan berhubungan dengan kelas-kelas dalam hierarki hewan, tetapi keduanya memiliki struktur terpisah.

Shotgun Surgery

Shotgun Surgery adalah code yang terkena dampaknya ketika perubahan kecil pada code memaksa perubahan besar pada banyak kelas atau modul yang berbeda. Ini mengindikasikan bahwa logika terkait secara fungsional tersebar di beberapa bagian code yang berbeda.
