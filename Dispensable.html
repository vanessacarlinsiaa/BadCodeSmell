Dispensable

Dispensable adalah sesuatu yang tidak berguna dan tidak diperlukan didalam sebuah code. Keberadaannya tidak akan membuat code menjadi lebih bersih, lebih efisien, dan lebih mudah dipahami. Sehingga, jika dihapus tidak akan mempengaruhi fungsionalitas dari program.

Comments

Comments adalah kondisi di mana code memiliki banyak comment yang seharusnya tidak perlu. Ini mungkin terjadi karena code yang kompleks atau sulit dipahami, sehingga pengembang merasa perlu untuk menambahkan banyak comment untuk menjelaskan apa yang dilakukan oleh code tersebut.

Data Class

Data class adalah class yang hanya berfungsi sebagai wadah untuk menyimpan data tanpa memiliki logika bisnis yang signifikan. Dispensable bad code smell terjadi ketika ada data class yang sebenarnya tidak diperlukan dan hanya menambah kompleksitas code tanpa memberikan nilai tambah yang nyata.

Dead Code

Dead code merujuk pada potongan code yang tidak pernah dieksekusi dalam aplikasi. Ini bisa berupa function, variabel, atau bahkan blok code yang tidak lagi digunakan atau diakses dalam aplikasi.

Duplicate Code

Duplicate Code adalah situasi di mana potongan code yang sama atau serupa muncul di beberapa bagian dari program. Hal ini memperumit pemeliharaan dan pengembangan code, karena perubahan yang dibutuhkan harus dilakukan di beberapa tempat, meningkatkan risiko kesalahan dan menyulitkan pemahaman code.

Lazy Class

Lazy class adalah sebuah class yang tidak memberikan banyak nilai tambah atau fungsionalitas yang cukup signifikan dalam program. class semacam itu sering kali hanya memiliki beberapa metode atau properti, dan tidak memiliki alasan yang kuat untuk keberadaannya.

Speculative Generality

Speculative Generality terjadi ketika sebuah code memiliki fitur atau function yang tidak digunakan saat ini, tetapi disertakan karena diperkirakan mungkin akan berguna di masa depan. Hal ini dapat menciptakan kompleksitas yang tidak perlu dan mempersulit pemahaman terhadap code.
